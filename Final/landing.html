System Overview

The goal of this website is to host a weather app. In this app, the user can input information that will accurately depict the weather in a given location, and store that information in a database. That database can then be queried for previous weather search history and directions. The TomTom API will be used for geolocation, and the results will then be sent in an API call to OpenWeatherMap, which will send results to the database.
